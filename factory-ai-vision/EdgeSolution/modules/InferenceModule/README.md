## Quick Start

1. Run Server

   ```bash
   make install

   python server.py
   # Or
   make run
   ```

## Developers

1. Install

   ```bash
   make install-dev
   ```

2. Tests

   ```bash
   make test
   ```

3. Coverage

   To run the tests, check your test coverage, and generate an HTML coverage report::

   ```bash
   make coverage
   ```
